# luz-lantern

## Conf

### Install

- git clone https://github.com/rodesousa/luz-lantern.git 
- cd luz-lantern
- ln -rs github.com $GOPATH/src/
- cd $GOPATH/src/github.com/rodesousa/lantern
- go install .
- $GOPATH/bin/lantern

### luz with vim

- Check install https://github.com/rodesousa/vim_conf.git
- In vim, :GoInstallBinaries
